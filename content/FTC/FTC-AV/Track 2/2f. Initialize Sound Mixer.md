
Supplies needed: None.
1. Tap the top-left corner of the screen, which should show a scene number and an "Initial" scene. ![[FTC/FTC-AV/Track 2/Media/IMG_4842 (1).jpg]]
2. In the menu that opens, select the FTC scene that corresponds with the type of event (single or dual-division), then touch "Recall". Confirm any pop-up that appears.
> [!important]- Selecting Dual-Division Scenes
> Be sure to select the scene that corresponds with the roles decided upon in the [[FTC/FTC-AV/Track 2/2e. Connect Speakers|previous stage]]. The primary mixer board should load `FTC DualDiv PRI`, and secondary should load `FTC DualDiv SEC`.

3. The controls will reset to the baseline state, including setting all volume to minimum.
Stage 2f is complete. [[FTC/FTC-AV/Track 2/2h. Sound Check and Gain Leveling|Proceed to stage 2g]].
